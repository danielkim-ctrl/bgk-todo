1 로그인하기

앱에 접속하면 팀별로 구분된 사용자 목록이 표시됩니다.

  1. 팀별로 구분된 사용자 목록에서 내 이름을 클릭합니다.
  2. 6자리 PIN 번호를 입력합니다. (PIN을 모르면 팀 관리자에게 문의)
  3. 로그인 버튼을 누르면 완료.
팁: PIN 번호는 설정 > 담당자 탭에서 관리자가 설정합니다. 최초 PIN은 관리자가 알려드립니다.

2 화면 구성 이해하기

로그인 후 기본 화면은 리스트 뷰입니다.

화면 영역 설명

영역설명
상단 헤더 (남색)로고, 팀 전환, 설정, 단축키, 사용자 정보
탭 바대시보드 / 리스트 / 캘린더 / 칸반 — 뷰 전환
왼쪽 사이드바검색, 프로젝트·담당자·우선순위·상태 필터
업무 추가 섹션직접 입력, AI 자동 입력, 템플릿 탭
업무 테이블프로젝트, 업무명, 상세, 담당자, 마감기한, 우선순위, 상태
팁: 사이드바 좌측 하단의 화살표를 클릭하면 사이드바를 접거나 펼 수 있습니다.

3 업무 추가하기

업무를 추가하는 3가지 방법이 있습니다. 리스트 뷰 상단의 "업무 추가" 영역에서 탭을 전환하여 사용합니다.

방법 1: 직접 입력

"+ 직접 입력" 탭을 선택하면 테이블 형태의 입력 행이 나타납니다.

▲ 직접 입력 탭 — 각 번호는 아래 설명과 대응됩니다. (클릭하면 확대)
  1. "+ 직접 입력" 탭을 클릭하여 입력 모드로 전환합니다.
  2. 좌측 상단의 "N개 저장" 버튼으로 입력한 모든 행을 한 번에 저장합니다.
  3. 프로젝트 드롭다운에서 프로젝트를 선택합니다. (상위 › 세부 프로젝트 트리)
  4. 업무내용 칸에 할 일을 입력합니다. (필수)
  5. 상세내용 텍스트를 클릭하면 상세 메모 팝업이 열립니다.
  6. 담당자를 드롭다운에서 선택합니다. (기본: 본인)
  7. → 마감기한(날짜 선택기), 우선순위(긴급/높음/보통/낮음), 반복(매일/매주/매월) 설정도 가능합니다.
  8. 우선순위를 선택합니다. 행 우측 끝의 ✓ 버튼으로 해당 행만 개별 저장합니다.
팁: "+ 행 추가" 버튼을 클릭하면 여러 업무를 한 번에 입력하고 일괄 저장할 수 있습니다.

방법 2: AI 자동 입력

"AI 자동 입력" 탭에서 텍스트나 파일을 분석하여 업무를 자동 생성합니다.

STEP 1 — 텍스트 입력 또는 파일 첨부

▲ AI 자동 입력 — 텍스트를 입력하고 "TODO 자동 생성"을 클릭합니다. (클릭하면 확대)
  1. "AI 자동 입력" 탭을 클릭합니다.
  2. 파일·이미지 첨부 — 이미지/PDF를 드래그하거나 클릭하여 업로드합니다. 스크린샷은 Ctrl+V로 바로 붙여넣기 가능합니다.
  3. 텍스트 영역에 업무 지시 내용을 자유롭게 입력합니다.
    예: "김대윤 — 계약서 검토 4/10까지, 박정찬 — 발주서 작성 긴급"
  4. 좌측 하단의 "TODO 자동 생성" 버튼을 클릭하면 AI가 업무를 분석합니다.

STEP 2 — 결과 확인 및 등록

AI 분석이 완료되면 아래에 결과 미리보기가 나타납니다.

▲ AI 결과 미리보기 — 일괄배정 드롭다운(⑤)과 등록 버튼(⑥)이 표시됩니다. (클릭하면 확대)
  1. 일괄배정 드롭다운(프로젝트, 담당자, 우선순위, 반복, 마감기한)으로 전체 항목을 한 번에 설정할 수 있습니다. 개별 항목을 클릭하여 업무명, 담당자, 날짜 등을 직접 수정할 수도 있습니다.
  2. 우측 상단의 "등록" 버튼을 클릭하면 체크된 항목이 업무로 생성됩니다. 체크박스를 해제하면 해당 항목은 등록에서 제외됩니다.
팁: AI가 추출한 업무명·담당자·마감일·우선순위는 모두 수정 가능합니다. 결과가 정확하지 않으면 텍스트를 더 구체적으로 작성해보세요.

방법 3: 템플릿 사용

"템플릿" 탭에서 미리 저장된 업무 세트를 한 번에 등록합니다.

▲ 템플릿 탭 — 검색/적용/새 템플릿 만들기. (클릭하면 확대)
  1. 업무 추가 영역에서 "템플릿" 탭을 클릭합니다.
  2. 검색창에서 템플릿을 검색합니다. 카테고리 칩, 팀 필터도 사용 가능합니다.
  3. 원하는 템플릿의 "적용" 버튼을 클릭합니다. → 미리보기 화면에서 기준일 설정, 일괄배정, 개별 수정 후 "N개 등록"을 클릭합니다.
  4. "+ 새 템플릿 만들기"로 자주 사용하는 업무 세트를 직접 저장할 수 있습니다.
팁: 즐겨찾기(★)를 클릭하면 자주 쓰는 템플릿이 목록 최상단에 고정됩니다. 템플릿은 "+ 새 템플릿" 버튼으로 직접 만들 수 있습니다.

4 업무 관리하기

업무를 수정, 완료, 삭제하는 방법입니다.

인라인 편집 (리스트 뷰)

▲ 리스트 뷰에서 셀을 클릭하면 바로 수정할 수 있습니다. (클릭하면 확대)
  1. 테이블에서 수정할 셀을 클릭합니다.
  2. 프로젝트, 담당자, 우선순위, 상태는 드롭다운으로 변경합니다.
  3. 마감기한은 날짜 선택기가 열립니다.
  4. 변경 사항은 자동 저장됩니다.

수정 모달

  1. 업무 행을 더블클릭하거나, 우클릭 메뉴에서 "편집"을 선택합니다.
  2. 수정 모달에서 모든 필드를 변경할 수 있습니다.
  3. 상세내용에서 리치 텍스트 편집(굵게, 기울임, 목록)이 가능합니다.
  4. 첨부 영역에서 파일/링크를 추가할 수 있습니다.
  5. "저장" 버튼을 클릭합니다.

우클릭 메뉴

▲ 업무 행을 우클릭하면 나타나는 메뉴 (클릭하면 확대)

업무 행을 마우스 우클릭하면 빠른 액션 메뉴가 나타납니다:

메뉴설명
편집수정 모달 열기
복제동일한 업무를 [복사]로 새로 생성
즐겨찾기즐겨찾기 토글
프로젝트 이동다른 프로젝트로 이동
삭제업무 삭제 (확인 후)

완료 처리

  1. 업무의 상태 열을 클릭하여 "완료"로 변경합니다.
  2. 또는 체크박스를 클릭합니다.
  3. 완료된 업무는 목록 하단 "완료됨" 섹션에 접혀서 표시됩니다.
팁: 실수로 삭제했다면 Ctrl+Z로 되돌릴 수 있습니다. 또는 헤더의 휴지통 버튼에서 복원할 수 있습니다.

5 뷰 전환

상단 탭 바에서 5가지 뷰를 전환할 수 있습니다. 리스트 뷰 내에서 리스트/메모 탭을 전환할 수 있습니다.

리스트 뷰

▲ 리스트 뷰 — 테이블 형태로 전체 업무를 관리합니다. (클릭하면 확대)

테이블 형태로 모든 업무를 한눈에 봅니다. 열 정렬, 인라인 편집, 다중 선택이 가능합니다.

메모 뷰 (스티키노트)

▲ 메모 뷰 — 포스트잇 스타일 카드로 업무를 관리합니다. (클릭하면 확대)

리스트 뷰 상단의 "메모" 탭을 클릭하면 전환됩니다. 업무가 포스트잇(스티키노트) 형태의 카드로 표시됩니다.

기능방법
카드 색상 변경카드 우측 상단 ··· 버튼 → 색상 팔레트 선택 (노랑, 핑크, 라벤더, 파랑, 회색)
업무명 편집카드 제목을 클릭하여 직접 수정
상세내용 편집카드 본문 영역을 클릭 → 리치텍스트 편집 (B/I/U, 목록, 색상)
프로젝트·상태 변경카드 하단의 드롭다운 클릭
완료 처리카드 하단 체크 버튼 클릭 → 카드가 반투명으로 변경
카드 추가우측의 "+ 추가하기" 카드 클릭
열 수 변경숫자 키 1~5 또는 상단 열 수 버튼
팁: 완료된 카드는 목록 하단에 접혀 표시됩니다. "완료됨" 영역을 클릭하면 펼쳐볼 수 있습니다.

캘린더 뷰

월간 캘린더에 마감기한 기준으로 업무가 표시됩니다. 우측에 "나의 할 일" 사이드바가 있습니다.

기능방법
날짜 클릭해당 날짜에 빠른 업무 추가
이벤트 클릭업무 상세 팝오버
드래그나의 할 일에서 섹션(오늘/내일/이번주) 간 이동 → 날짜 자동 변경
뷰 전환일/주/월/N일/일정 5가지 서브 뷰

칸반 뷰

▲ 칸반 뷰 — 카드를 드래그하여 상태를 변경합니다. (클릭하면 확대)

대기/진행중/검토/완료 4개 컬럼으로 업무 상태를 시각화합니다. 카드를 드래그앤드롭으로 다른 컬럼에 놓으면 상태가 자동 변경됩니다.

대시보드

▲ 대시보드 — 팀 업무 현황, 브리핑, 데일리 활동을 한눈에 봅니다. (클릭하면 확대)

팀 전체 업무 현황을 한눈에 봅니다. 상단에 내 업무 현황 카드, 오늘의 브리핑, KPI 카드, 데일리 활동/멤버별/프로젝트별 통계가 표시됩니다.

6 필터와 검색

사이드바에서 원하는 업무만 골라볼 수 있습니다.

검색

사이드바 상단 또는 리스트 뷰 검색창에 키워드를 입력하면 업무명 + 상세내용에서 실시간 검색됩니다.

필터 사용

  1. 사이드바에서 프로젝트, 담당자, 우선순위, 상태 섹션을 확인합니다.
  2. 원하는 항목을 클릭하면 필터가 적용됩니다. (여러 개 동시 선택 가능)
  3. 다시 클릭하면 해제됩니다.
  4. "전체"를 클릭하면 해당 섹션 필터가 초기화됩니다.

프로젝트 트리 필터

세부 프로젝트가 있는 상위 프로젝트를 클릭하면 하위 전체가 함께 선택됩니다. 세부 프로젝트를 개별 클릭하면 해당 세부만 필터링됩니다.

팀별 그룹

전체보기 모드에서 팀이 2개 이상이면, 프로젝트와 담당자가 팀별 그룹 헤더로 구분되어 표시됩니다. 그룹 헤더를 클릭하면 접기/펼치기가 가능합니다.

팁: 필터를 저장해두면 나중에 한 클릭으로 불러올 수 있습니다. 리스트 뷰 상단의 "저장" 버튼을 활용하세요.

7 프로젝트와 세부 프로젝트

프로젝트는 2단계 계층 구조를 지원합니다. 설정 > 프로젝트 탭에서 관리합니다.

▲ 설정 > 프로젝트 탭 — 상위/세부 프로젝트, 색상, 팀 배정을 관리합니다. (클릭하면 확대)

프로젝트 구조

레벨예시설명
상위 프로젝트만화웹툰2026아시아큰 프로젝트 단위
세부 프로젝트└ 중국, └ 일본, └ 태국국가별/단계별 하위 분류

프로젝트 관리 (설정)

  1. 헤더의 설정(톱니바퀴) 버튼을 클릭합니다.
  2. 프로젝트 탭을 선택합니다.
  3. 상위 프로젝트 아래 "+ 세부 프로젝트 추가"를 클릭합니다.
  4. 색상 도트를 클릭하면 프로젝트 색상을 변경할 수 있습니다.
  5. 눈 아이콘(👁)을 클릭하면 프로젝트를 숨깁니다. (모든 뷰에서 사라짐)

새 프로젝트 추가

설정 하단의 "새 프로젝트" 영역에서 이름, 색상, 소속 팀을 선택하고 "추가" 버튼을 클릭합니다.

8 팀 관리

설정 > 팀 탭에서 팀을 생성하고 멤버를 배정합니다.

팀 전환

헤더의 팀 드롭다운에서 보고 싶은 팀을 선택하면, 해당 팀의 프로젝트와 멤버만 표시됩니다. "전체보기"를 선택하면 모든 팀을 볼 수 있습니다.

권한 체계

역할별로 사용할 수 있는 기능이 다릅니다. 설정 > 권한 탭에서 관리자가 세부 권한을 커스텀할 수 있습니다.

10 자주 묻는 질문 (FAQ)

Q: 실수로 업무를 삭제했어요.

A: Ctrl+Z로 되돌리거나, 헤더의 휴지통 아이콘에서 복원할 수 있습니다.

Q: 다른 사람의 업무를 수정할 수 없어요.

A: 뷰어 역할은 본인 업무만 수정 가능합니다. 타인 업무 수정이 필요하면 팀 관리자에게 편집자 이상 역할을 요청하세요. (설정 > 권한 탭에서 역할별 권한 확인 가능)

Q: 프로젝트가 너무 많아서 보기 불편해요.

A: 설정 > 프로젝트에서 완료된 프로젝트의 눈 아이콘을 클릭하여 숨길 수 있습니다. 모든 뷰에서 사라집니다.

Q: AI 자동 입력이 작동하지 않아요.

A: 설정 > API 키 탭에서 Anthropic API 키가 올바르게 입력되었는지 확인하세요. 키가 맞는데도 안 되면 API 사용량 한도를 확인해보세요.

Q: 모바일에서도 사용할 수 있나요?

A: 네. 모바일 브라우저에서 접속하면 자동으로 모바일 레이아웃이 적용됩니다. 하단 탭 바와 카드형 리스트가 제공됩니다.

Q: 데이터는 어디에 저장되나요?

A: Google Firebase Firestore에 실시간으로 저장됩니다. 여러 기기에서 동시에 접속해도 데이터가 자동 동기화됩니다.

Q: 반복 업무는 어떻게 설정하나요?

A: 업무 수정 모달에서 반복 드롭다운을 클릭하여 매일/매주/매월 중 선택합니다. 직접 입력 행에서도 반복 컬럼에서 설정 가능합니다.

Q: PIN 번호를 잊어버렸어요.

A: 팀 관리자에게 문의하세요. 관리자는 설정 > 담당자 탭에서 해당 멤버의 PIN을 확인하거나 변경할 수 있습니다.

Q: 업무를 다른 프로젝트로 옮기고 싶어요.

A: 리스트 뷰에서 해당 업무의 프로젝트 셀을 클릭하여 드롭다운에서 변경하거나, 우클릭 메뉴 > "프로젝트 이동"을 사용하세요.

Q: 여러 업무를 한 번에 삭제하고 싶어요.

A: 리스트 뷰에서 업무 왼쪽의 체크박스를 여러 개 선택하면 상단에 일괄 작업 바가 나타납니다. 거기서 삭제, 상태 변경, 담당자 변경 등을 한 번에 할 수 있습니다. Shift+클릭으로 범위 선택도 가능합니다.

Q: 업무 수정 시 빠르게 저장하는 방법이 있나요?

A: 수정 모달에서 Ctrl+Enter를 누르면 바로 저장됩니다.

Q: 캘린더에서 업무 날짜를 바꾸고 싶어요.

A: 캘린더 뷰의 우측 "나의 할 일" 사이드바에서 업무를 드래그하여 오늘/내일/이번 주 섹션으로 이동하면 마감일이 자동 변경됩니다.

Q: 완료된 업무가 너무 많아서 지저분해요.

A: 리스트 뷰 하단의 "완료됨" 섹션은 기본적으로 접혀 있습니다. 클릭하면 펼쳐서 볼 수 있고, 평소에는 접혀있어 작업 중인 업무에 집중할 수 있습니다.

Q: 사이드바 필터를 저장할 수 있나요?

A: 네. 원하는 필터를 설정한 후 리스트 뷰 상단의 "저장" 버튼을 클릭하면 이름을 지정하여 저장됩니다. 저장된 필터는 언제든 한 클릭으로 불러올 수 있습니다.

Q: 다른 팀의 업무를 볼 수 있나요?

A: 관리자 역할만 가능합니다. 헤더의 팀 드롭다운에서 다른 팀을 선택하거나 "전체 보기"를 선택하세요. 편집자와 뷰어는 본인 소속 팀만 볼 수 있습니다.

Q: 업무에 파일을 첨부할 수 있나요?

A: 네. 업무 수정 모달 하단의 "첨부" 영역에서 "+ 파일/링크 추가"를 클릭하면 파일명과 URL을 입력할 수 있습니다.

Q: 새로고침하면 데이터가 사라져요.

A: 정상적으로는 Firebase에 자동 저장됩니다. 데이터가 사라진다면 네트워크 연결 상태를 확인하세요. 오프라인 상태에서 입력한 데이터는 연결 복구 시 동기화됩니다.
