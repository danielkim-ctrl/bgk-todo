3 조건부 표시 규칙

팀 수에 따라 그룹 헤더 표시 여부가 달라집니다.

조건사이드바 표시이유
소속 팀 1개그룹 헤더 없음 — 현재와 동일불필요한 UI 제거
소속 팀 2개 이상 + 전체보기팀별 그룹 인덱스 표시팀 경계 구분 필요
특정 팀 선택 상태그룹 헤더 없음 — 해당 팀만 표시이미 팀 필터링됨
팀에 미연결 프로젝트 있음"미배정" 그룹 하단 표시누락 방지
접힌 그룹 안에 활성 필터헤더에 ● 파란 점 표시"이 안에 선택된 항목 있음" 알림

5 인터랙션 정리

동작결과
팀 그룹 헤더 클릭해당 팀 소속 프로젝트/담당자 접기/펼치기
개별 프로젝트 클릭기존과 동일 — 해당 프로젝트만 필터 토글
개별 담당자 클릭기존과 동일 — 해당 담당자만 필터 토글
"전체" 클릭기존과 동일 — 모든 필터 해제
헤더 팀 전환특정 팀 선택 → 그룹 헤더 없이 해당 팀만 표시
