1. 텍스트 입력 영역 — Before vs After

현재 (Before)

rows=3, 높이 약 72px, resize: vertical만 가능

문제점:
  • 3줄 높이 → 긴 회의록 붙여넣으면 내용 확인 불가
  • resize가 오른쪽 하단(↘)만 가능 → 세로만 늘려야 할 때 불편
  • 글자 크기 12px → 장문 읽기 불편
  • placeholder가 한 줄만 보임
개선 (After)

min-height: 120px, resize: none + 하단 전체 드래그 핸들, focus 시 보라색 글로우

0자
개선점:
  • 기본 높이 180px → 내용을 한눈에 확인 가능
  • 네이티브 resize 제거 → 하단 전체 영역을 잡고 세로 크기 조절 (커스텀 드래그 핸들)
  • 포커스 시 보라색 글로우로 입력 중 상태 표시
  • 우측 하단 글자 수 카운터 추가
  • placeholder에 예시가 충분히 보여 안내 역할

3. 개선 항목 상세

1 텍스트 입력 영역 확대 + 하단 전체 드래그 핸들 High
현재: rows=3 (≈72px), resize: vertical (오른쪽 하단 모서리만 잡아야 함)
개선: min-height: 180px, 네이티브 resize 제거 → 하단 전체 영역을 드래그하여 세로 크기 조절
이유: 회의록이나 카카오톡 대화를 붙여넣으면 10~30줄인 경우가 많음. 오른쪽 하단 모서리만 잡는 것은 찾기 어렵고 불편함. 하단 전체를 잡을 수 있어 훨씬 직관적.
2 포커스 시 시각적 피드백 강화 Medium
현재: 테두리 색상 변화 없음, outline: none만 적용
개선: focus 시 border-color: #7c3aed + box-shadow: 0 0 0 3px rgba(124,58,237,.1)
이유: 입력 중인 필드가 어디인지 한눈에 알 수 있어야 함. Google 서비스의 파란 글로우 패턴을 보라색으로 적용.
3 글자 수 카운터 추가 Low
현재: 입력량 피드백 없음
개선: 우측 하단에 N자 표시 (실시간 업데이트)
이유: AI 분석의 입력 길이 제한이 있을 수 있고, 사용자가 얼마나 입력했는지 감각적으로 파악 가능.
4 빠른 입력 힌트 칩 추가 Medium
현재: placeholder 텍스트로만 예시 안내 → 입력 시작하면 사라짐
개선: "회의록 예시", "카카오톡 예시", "업무보고 예시" 칩 클릭 → 샘플 텍스트 자동 채움
이유: 처음 쓰는 사용자가 "어떤 형식으로 써야 하나?" 고민 없이 바로 체험 가능. 온보딩 효과.
5 파일 첨부 영역 시각적 강화 Low
현재: 작은 아이콘 + 텍스트, 클릭 영역이 좁음
개선: 아이콘을 40px 원형 배경 안에 배치, 전체 영역 터치 타깃 확대, hover 시 보라색 전환
이유: 드래그 앤 드롭의 대상 영역이 명확히 보여야 사용자가 파일을 쉽게 첨부할 수 있음.
6 결과 미리보기 행 가독성 향상 Medium
현재: 배지 높이 22px, 폰트 10px, 간격 좁음 → 정보가 밀집되어 읽기 어려움
개선: 배지 높이 26px, 폰트 11px, 행 패딩 확대, 업무명 14px 강조
이유: AI가 추출한 결과를 수정하기 전에 빠르게 훑어볼 수 있어야 함. 현재는 너무 작아서 하나씩 눈을 가까이 대야 확인 가능.
7 일괄배정 버튼에 텍스트 라벨 추가 Medium
현재: 아이콘만 있는 버튼 → 무엇을 하는 버튼인지 hover 해봐야 알 수 있음
개선: 아이콘 + "프로젝트", "담당자", "우선순위" 텍스트 라벨 병기
이유: 처음 쓰는 사용자가 아이콘만으로 기능을 이해하기 어려움. 라벨 추가로 발견 가능성 100% 확보.

4. 구현 우선순위

# 항목 심각도 난이도 영향 범위
1 텍스트 입력 영역 확대 + 하단 전체 드래그 High 쉬움 AddTodoSection textarea 스타일만 변경
2 포커스 시각적 피드백 Medium 쉬움 onFocus/onBlur 핸들러 추가
3 결과 미리보기 가독성 향상 Medium 보통 aiSelBase 스타일 + 결과 행 패딩/폰트 조정
4 일괄배정 버튼 라벨 추가 Medium 쉬움 버튼 텍스트만 추가
5 빠른 입력 힌트 칩 Medium 보통 AddTodoSection에 칩 컴포넌트 추가
6 글자 수 카운터 Low 쉬움 textarea 하단 span 추가
7 파일 첨부 영역 시각 강화 Low 쉬움 드롭존 스타일만 변경
