필터 저장 기능 — 3가지 설계안 데모
각 탭을 클릭하여 인터랙티브 데모를 확인하세요
안 1 — 사이드바 내 "저장된 필터" 섹션
현재 필터를 북마크처럼 저장. 사이드바 최상단에 별도 섹션으로 노출. Google Tasks 라벨 패턴.
1
필터 적용 후 사이드바 상단 "+ 현재 필터 저장" 클릭
2
이름 입력 후 저장 → "저장된 필터" 섹션에 항목 추가
3
저장된 항목 클릭 → 해당 필터 즉시 적용

안 2 — 뷰 탭 드롭다운
필터 조합을 이름이 붙은 "뷰"로 저장. 필터 칩 바 좌측에 드롭다운으로 뷰 전환. Google Sheets 필터 보기 패턴.
1
필터 적용 후 칩 바 우측 [+ 뷰로 저장] 클릭 → 이름 입력
2
좌측 뷰 선택 드롭다운에 항목이 추가됨
3
드롭다운에서 뷰 선택 → 필터 즉시 전환

안 3 — 칩 핀 방식 권장
현재 필터를 핀으로 고정. 저장된 필터는 노란 칩으로 칩 바 앞쪽에 항상 표시. 전 뷰 공통 동작. Gmail 저장 검색 + Linear 뷰 패턴.
1
필터 적용 → 칩 바 우측 [📌 저장] 버튼 클릭
2
이름 입력 → 칩 바 앞쪽에 노란 핀 칩으로 고정
3
핀 칩 클릭 → 해당 필터 전환 / hover → [×] 삭제 노출
