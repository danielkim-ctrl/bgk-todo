자주 반복되는 업무 묶음을 템플릿으로 저장해두고, 필요할 때 꺼내서 담당자·날짜만 바꿔 한 번에 등록하세요.

새 템플릿 만들기
기준일 담당자 일괄

기능 흐름 요약

1. 템플릿 만들기
반복되는 업무 묶음에 이름 붙여 저장
업무명 + 우선순위 + 상대 마감일(D+N)
2. 적용하기
기준일 입력 → 날짜 자동 계산
담당자 일괄배정 → 미리보기 확인
3. 등록
업무가 한 번에 생성됨
AI 결과와 동일한 미리보기 UI 재활용

상대 날짜(D+N) 개념

템플릿의 마감일은 절대 날짜가 아닌 상대 날짜로 저장됩니다. 적용 시 기준일을 입력하면 자동으로 실제 날짜가 계산됩니다.
